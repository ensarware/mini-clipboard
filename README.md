# mini-clipboard
Just a simple clipboard app built while learning Rust. It runs in the background, keeps a basic text history, and opens with a hotkey. Nothing fancy.

## What it does

- Saves text you copy (`Ctrl + C`).
- Opens a simple window when you press `Ctrl + Shift + V`.
- Has a basic search bar to find old text.
- Double-clicking an item copies it back to your clipboard.
- Saves data to a local file so it doesn't disappear when closed.
